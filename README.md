# Mail Calibration Tool (Excel VBA)

AIを活用したビジネスメールの校正・タイポチェックを効率化するためのExcel VBAツールです。

## 📋 概要
このツールは、ビジネスメールの送信前に誤字脱字をチェックし、生成AI（Ollama, Qwen, Gemmaなど）へ入力するための「校正用プロンプト」を自動生成します。
個人の実務効率化プロジェクトとして、VBAによる自動化とAI利活用を組み合わせて開発しました。

## ✨ 主な機能
- **自動プロンプト生成**: 入力されたメール本文に基づき、AIが理解しやすい校正依頼文を自動作成します。
- **タイポチェック補助**: 独自のロジックにより、誤字脱字の可能性がある箇所をハイライトします。
- **ワンクリック操作**: Excel上で全てのフローが完結するため、作業を中断することなく校正が可能です。

## 🛠 技術スタック
- **Language**: Excel VBA
- [cite_start]**Used AI Context**: Ollama, Qwen, Gemma (Local LLM) 
- **Platform**: Microsoft Excel

## 🚀 使い方
1. リポジトリから `mailcheck_001.xlsm` をダウンロードします。
2. ファイルを開き、「マクロを有効にする」を選択してください。
3. 指定のセルにメール本文を貼り付け、実行ボタンを押すと校正用プロンプトが出力されます。

## 📝 開発の背景
[cite_start]データサイエンスの学習や個人の投資活動  の中で、外部とのコミュニケーションをより正確かつ迅速に行うために開発しました。
